# Project Memory (Claude Code)

This repository is a Next.js + TypeScript project using shadcn/ui components.

## Where to look first

- Project rules live under: `.claude/rules/`
- Subagents live under: `.claude/agents/`

## Expectations

- Prefer small, incremental changes.
- Keep conventions consistent with the existing codebase.
- Do not modify secrets or environment files.
- If unsure about a high-impact change, ask first.

## Key directories (once generated)

- `src/app/` (Next.js App Router)
- `src/components/` (shared components)
- `src/lib/` (utilities)
